LoadPack

A smart backpack that measures load and balance using sensors + Bluetooth.

How it works Demo App Team

Know your load. Improve comfort. Stay safe.

LoadPack uses load sensors on the straps to estimate total weight and left/right balance. Data is sent via Bluetooth to a mobile app for live display and logging.

See Demo How It Works

Project status: Building + testing

Due: Jan 13

Tech: ESP32 + HX711 + Load Cells + BLE + App

How it works

  1. Two load cells measure strap tension (left + right).
  2. HX711 amplifiers convert sensor signals to readable values.
  3. ESP32 calculates left/right/total and broadcasts data over BLE.
  4. Mobile app displays and records the data.
