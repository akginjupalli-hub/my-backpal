Two strap sensors estimate left/right balance and total load. The ESP32 broadcasts data over Bluetooth to a phone app (and this website demo).

How it works

  1. Load cells measure strap tension (left + right).
  2. HX711 amplifies signals so the ESP32 can read them.
  3. ESP32 calculates L/R/Total and broadcasts via BLE notifications.
  4. App + Website show live data and log it.
